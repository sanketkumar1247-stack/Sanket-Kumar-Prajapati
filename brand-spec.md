# Brand system — Sanket Kumar Prajapati portfolio

Soft medical/academic portfolio direction matched to the Dribbble doctor reference: cool blue-grey canvas, pill navigation, circular portrait with floating icon badges, oversized split name type.

## Posture

1. Soft blue shapes + faint grid behind hero; white card stage with large radius.
2. Pill nav (light blue track, white active chip).
3. Circular portrait centered; name split left/right in bold blue sans.
4. Floating white badges (plus, book) with soft shadow — no emoji.
5. Cards: white, soft shadow, rounded corners; accent for outlets/meta only.
